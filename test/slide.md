# 一般的な機能

## `\SLyDIFi;`{=satysfi} におけるフレーム

- フレーム：スライド資料の1ページ1ページに値するもの
- `\SLyDIFi;`{=satysfi} では3種類のフレームを区別する
    - 見出し：スライド全体の題目，発表者名などを載せるフレーム
    - セクション見出し：セクションのタイトルを載せる
    - 本文：通常のフレーム

## テキストの記述

通常のマークダウン記法を大体サポートしている。

- 数字のない箇条書き
    + 箇条書きの入れ子
1. 数字のある箇条書き
    2. ここでも入れ子にできる
- 数字のある箇条書きとない箇条書きの親子は未サポート
- *強調* と **強い強調** は同じ組版になる
- `~~打ち消し線~~` は未実装

`\SATySFi;`{=satysfi}コードを埋め込むと文字色の変更などより複雑な記述もできる。

- `\text-color(){}`: ``\text-color(Color.of-css `#42883B`){文字色変更}``{=satysfi}

## 図表の貼り付け

* 通常通りのマークダウン記法で画像を読み込むことができる。中央揃えで表示される。

![デフォルト幅での画像](figure.jpg)

* デフォルトの幅はスライドの用途だとかなり狭い
    * 幅を別途指定する以下のように表示できる

![300ptの画像](figure.jpg){width=300pt}

## より高度な図表の貼り付け

```{=satysfi}
+fig-on-right(dummy-box 100pt 200pt)<
```

* 内部では`FigBox`を使って画像の読み込みなどを実現している。
* `\SATySFi;`{=satysfi}コードを埋め込めばかなり色々なことが実現できる。
    * `dummy-box`：指定されたサイズのダミーボックス
    * `hmargin`：水平方向に指定された量の余白を付ける
    * `vconcat`：鉛直方向に図を結合
    * `\fig-inline`：テキスト中 `\fig-inline?:(0.5)(dummy-box 60pt 30pt |> hmargin 5pt);`{=satysfi} に画像を出力
    * `+fig-on-right`：画像を右に，本文を左に配置

```{=satysfi}
>
```

# `\SLyDIFi;`{=satysfi} でオーバーレイ

## オーバーレイの基礎   {overlay=3}

オーバーレイは、同じスライドに対して「レイヤーを上から重ねる」かのように順々にアイテムを表示できる機能。マークダウンにそんな機能はないのでPandoc経由でこの機能を使うには`\SATySFi;`{=satysfi}コードを埋め込む必要がある。

```{=satysfi}
+ghost(only 1)<
```
- 表示を一方的に追加していくだけでなく、特定の番号でのみアイテムを削除したり、書式を変更したりすることも可能。
- 「何枚目のときに表示するか否か」を表す `int -> bool` 型の関数を用いて指定する。
    - 但し大抵の用途では次のスライドに示す `only` や `from` 関数を使う範囲で十分

```{=satysfi}
>
```

```{=satysfi}
+ghost(from 2)<
```
- `+ghost(only 1)<>` とすれば、1枚目のみで出現するブロックが作れる。
- `+ghost(from 2)<>` とすれば、2枚目以降で出現するブロックが作れる。
```{=satysfi}
>
```

```{=satysfi}
+ghost(from 3)<
```
- `\SLyDIFi;`{=satysfi} は箇条書きのオーバーレイもサポートしているが、このwriterでは未サポート。
```{=satysfi}
>
```
